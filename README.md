# Classification
Этот обучающий прект наглядно демонстрирует все этапы решения классической DS задачи классификации. В качестве примера взята задача кредитного скоринга, датасет взят отсюда: (https://www.kaggle.com/code/olzhaskolzhabay/bank-marketing-analysis-prediction)

Проект включает в себя следующие этапы:

    Проверка на корректность и качество данных (файл EDA.ipynb)
    Детальное изучение признаков и таргета (файл EDA.ipynb)
    Изучение взаимосвязей между признаками (файл EDA.ipynb)
    Препроцессинг данных (файл baseline.ipynb):
        Масштабирование
        Кодирование категориальных переменных
        Приведение данных к единому типу
    Построение базовой модели и подсчет метрик (файл baseline.ipynb)
    Преобразования данных, тестирование с помощью кросс-валидации. Выбор лучших преобразований для улучшения предсказаний (файл improvements.ipynb)
    Подбор гиперпараметров разными методами (файл hyper_param_tuning.ipynb)
